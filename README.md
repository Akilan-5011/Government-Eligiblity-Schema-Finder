🚀 AI-Based Government Scheme Eligibility and Recommendation System

 📌 Overview

The AI-Based Government Scheme Eligibility and Recommendation System is a web application that helps citizens easily find government welfare schemes they are eligible for based on their personal details.

🎯 Problem Statement

Citizens struggle to identify and access eligible government schemes due to lack of awareness, complex eligibility criteria, and absence of a unified intelligent system.

💡 Solution

This system collects user details such as age, income, occupation, and location, and automatically recommends suitable government schemes using an intelligent eligibility engine.

🛠 Tech Stack

Node.js, Express.js, MongoDB, HTML5, CSS3, Vanilla JavaScript
 ✨ Key Features

* 🔍 Eligibility-based scheme recommendation
* 🎙 Voice search for easy access
* 📷 Document upload with OCR support
* 🔎 Smart search and filtering
* ❤️ Bookmark / favorite schemes
* ⏰ Deadline reminders
* 🤖 Chatbot assistant
* 📊 Admin dashboard and analytics
* 🌗 Dark mode support

🌍 Impact

This project simplifies access to government welfare schemes, improves awareness, and helps citizens—especially in rural areas—benefit from available opportunities.

🏆 Outcome

A centralized and intelligent platform that makes government schemes easy to discover, understand, and access.
